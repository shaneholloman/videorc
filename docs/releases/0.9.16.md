# Videorc 0.9.16 Beta 1

Videorc 0.9.16 ships the **OBS import wizard + viewer counts**, **native
Twitch OAuth** (first build baking `VIDEORC_BUNDLED_TWITCH_CLIENT_ID`),
and the **record+stream A/V fix** (Plan 023).

## Scope

- **OBS import** (vault plan, O1–O6, `a4183d58`→`99825f9f`) — Settings
  wizard with the truthful 3-verdict report before apply; main-process
  reader, pure renderer mapper, atomic merge; stream key rides the
  manualKey→secret-ref migration; fresh-profile Studio nudge;
  `smoke:obs-import`.
- **Live viewer counts** (V1 `a4466bc3`, V2 `f0b3e865`) — 30s
  YouTube/Twitch sampler, samples persisted per session
  (`stream-viewers` logs), "N watching" chip in Comments.
- **Twitch OAuth** (`9d27760e`, `c8bf97f2`) — Public client type (id-only,
  no secret anywhere), localhost redirect form for Twitch's console;
  BUNDLED client id bakes at build from the release env.
- **Record+stream A/V** (Plan 023, `dfe89df1`→`7cdf4a40`) — MpegTs (real
  encoder PTS) default on every leg; LVF2 root causes fixed (probe
  starvation, codec-tag [27]→flv, fatal refused outputs → fifo-muxer
  legs); duplicate-PTS analyzer gate; both-leg smoke analysis;
  `recording-degraded` mid-session warning; URL-attributed dead legs.
