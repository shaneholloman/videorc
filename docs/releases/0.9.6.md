# Videorc 0.9.6 Beta 1

Videorc 0.9.6 is the **library + comments + theme** release: the Library
rebuilt as a recordings manager, the Comments window upgraded to an
all-platform reader with on-stream highlight and send-to-all, and the
black-glass dark / porcelain light theme.

## Scope

- **Library rewrite** (L1–L6) — recordings-manager table: posters extracted
  at finalize + lazy idle-aware backfill, served over the backend HTTP
  server; live file sizes; derived scene chips; parameterized list depth +
  storage totals; rename/duplicate/import RPCs; delete = renderer moves
  files to the system Trash first, backend never unlinks; pinned
  header/toolbar with only the table scrolling; Publish rail shows the same
  posters + scene chips. The always-on Status/quality column was removed
  (Check quality / Repair / Restore stay in the row kebab). New
  `pnpm smoke:session-ops` + poster assert in `smoke:dev`. Acceptance:
  `docs/acceptance/2026-07-05-library-rewrite.md`.
- **Comments window upgrade** (S1–S6) — platform glyphs + allowlisted avatar
  cache on every message; click-to-highlight renders the comment on the
  stream via a dedicated overlay slot (coexists with captions); send fan-out
  posts one reply to every capable platform (YouTube in scope; Twitch needs
  the user:write:chat scope — pre-existing connections get a
  reconnect-required prompt). Acceptance:
  `docs/acceptance/2026-07-05-comments-window-upgrade.md`.
- **Black glass theme** (S0) — dark base retuned to true black glass with
  polished edges; light becomes the porcelain twin; brand red unified on the
  LED family and kept semantic (record/live/destructive only).
