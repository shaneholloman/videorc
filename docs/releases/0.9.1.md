# Videorc 0.9.1 Beta 1

Videorc 0.9.1 is the first beta with **in-app updates**. It builds on the 0.9.0
beta 1 signed macOS candidate.

> **Known issue (fixed in 0.9.2):** camera and microphone capture are DEAD in
> this packaged build even when the user grants permission — the hardened-runtime
> signature lacked the AV device entitlements (`com.apple.security.device.camera`
> / `device.audio-input`), which a TCC grant cannot override. Screen recording
> was unaffected (no entitlement required). See `docs/releases/0.9.2.md`.

## Scope

- **In-app updater** — Settings → About & updates: "Check for updates →
  download (with progress) → Restart & install", with install blocked while a
  recording/stream is live. Backed by the electron-updater feed served at
  videorc-web `/api/updates/*`.
- App version is now surfaced in Settings.
- Everything in 0.9.0 beta 1: local screen/camera/microphone recording, native
  detached preview, source and layout controls, bundled backend + native preview
  helper + FFmpeg, authenticated download through Videorc Web.

## Update path

Installed 0.9.0 clients (whose baked feed URL points at the current web host) are
offered 0.9.1 once the signed + notarized build is uploaded — `package.json`
`version` 0.9.0 → 0.9.1 is the electron-updater compare key. See
[release-runbook.md](release-runbook.md) for the cut/upload steps.
