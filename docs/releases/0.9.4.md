# Videorc 0.9.4 Beta 1

Videorc 0.9.4 is the **docked preview** release: the detached native preview
can be stuck into the Studio slot as a child window of the app, plus scene-nudge
and AI-transcript polish.

## Scope

- **Docked ("stick") preview** — opt-in mode: the preview window becomes an
  AppKit child of the main window glued over the Studio slot; renderer is never
  in the movement path (window-relative slot reports + main-process bounds
  composition; dock epochs reject stale reports). Every hide has a stated
  reason; docked surfaces never `elevated`; dock/undock is placement-only.
  Probe scenario folded into recording-studio gates. Acceptance:
  `docs/acceptance/2026-07-04-docked-preview-stick-acceptance.md`.
- **Scene nudge** — arrow keys escape the corner-snap magnet; disabled arrows
  explain themselves.
- **AI transcript sourcing** — uses the recording file that exists; explains
  when none does.
- **Black glass** — a hint of true translucency behind the window (dark mode).
- **Studio chrome cleanup** — removed the page header/warning + duplicate
  transport controls.
- Open-source prep (AGPL-3.0 LICENSE + public README) — repo stays private
  until launch day.
