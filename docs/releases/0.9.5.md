# Videorc 0.9.5 Beta 1

Videorc 0.9.5 is the **caption-integrity + polish batch** release: the caption
carry-over root fix, the docked-preview tab-switch fix, background upload
repair, the calm Studio, and end-to-end Premium enforcement for
multistreaming.

## Scope

- **Caption carry-over root fix** (C1/C2) — per-session caption floor in the
  renderer + backend overlay clear at start_session + stale-epoch update
  suppression. Acceptance: `docs/acceptance/2026-07-04-caption-carry-over-fix.md`.
- **Post-0.9.4 UI fix batch** (F1–F8) — docked preview leaves the screen with
  its tab (explicit placement push + CGWindowList probe assert) and the
  placeholder is output-aspect-sized; `videorc-asset://` protocol fixes
  "Missing" background imports; Studio blocked reasons move inline (yellow
  banner removed); captions helper copy gone when on; Scene sources panel
  removed + compact full-canvas inspector; Library issue badge + tooltip;
  sidebar update chip; live mixer from session-captured frames. Acceptance:
  `docs/acceptance/2026-07-05-post-094-ui-fix-batch.md`.
- **Multistream Premium enforcement** — account-verified entitlement (from
  /api/ai/capabilities) hydrates the backend snapshot at boot/sign-in/refresh,
  clears on sign-out, 24h staleness ceiling, fail-closed everywhere;
  `VIDEORC_PREMIUM_FEATURES=0` forces basic in dev; locked destination rows
  show a Premium lock chip that opens the upgrade page.
- Tab content top-aligned with the sidebar start.
- Everything in 0.9.4 (docked "stick" preview).
