# Videorc 0.9.15 Beta 1

Videorc 0.9.15 is the **launch-domain flip** release plus the day's fix
batch.

## Scope

- **Host flip** (`66d160ff`) — every baked origin →
  `https://www.videorc.com`: electron-builder `publish.url` (update feed),
  `videorc-web-links.ts` (account/login/auth-bridge/premium/billing/
  changelog), Rust `PRODUCTION_API_BASE_URL`, changelog-email default.
  WWW is load-bearing (apex 307s everything; reqwest strips Authorization
  cross-host). ⚠️ Keep the Vercel host's `/api/updates/*` serving until
  builds ≤0.9.14 age out.
- **Notes/Comments/Captions always-on-top fix** (`3552e38c`) —
  `skipTransformProcessType`; the activation-policy flip was hiding every
  other app window while an aux window with always-on-top was open.
- **Chrome fixes** (`e4ac818c`) — dropdown menus size to content (Library
  "…" menu was trigger-width and wrapped every label); Comments window
  header insets 78px clear of the traffic lights.
- **Studio health** (`94598978`) — the red "requires native CAMetalLayer"
  Blocked state removed outright; non-native transports warn as
  "Fallback", absent transports are never a health state (the preview
  window's presenting watch owns preview-path health).
