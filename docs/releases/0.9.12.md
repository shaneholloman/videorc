# Videorc 0.9.12 Beta 1

Videorc 0.9.12 is the **rotated YouTube OAuth secret** release (Plan 020,
GitGuardian incident). No product changes; the entire point is that the
packaged backend embeds the rotated
`VIDEORC_BUNDLED_YOUTUBE_CLIENT_SECRET` so the old (leaked) secret can be
disabled in Google Cloud Console without stranding users.

## Scope

- **Rotated Google OAuth desktop client secret** — the release env
  (`~/.videorc-release.env`) was rotated by the owner (fingerprint
  verified: no longer matches the leaked value); this build compiles the
  new secret in via `option_env!`.
- **Exact-secret release gate** — `release:validate:macos` now proves the
  packaged `videorc-backend` contains the exact
  `VIDEORC_BUNDLED_YOUTUBE_CLIENT_SECRET` value from the release
  environment via an in-process binary check (never printed, never passed
  as a command argument), replacing the committed `GOCSPX-` prefix grep.

## Notes

- Order is load-bearing: ship + verify this build FIRST, then disable the
  old secret. Existing installed builds embed the old secret and lose
  YouTube OAuth the moment it is disabled — that is expected and is why
  the changelog tells users to update.
- Git history still contains the leaked value; rewrite is optional and
  separately coordinated (Plan 020 Phase 5).
